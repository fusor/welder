• Foreman fusor_v21

Satellite API v1 is deprecated. Please use v2. If you still need to use v1, you may do so by either passing ‘version=1’ in the Accept Header or using api/v1/ in the URL.

Resources

Customer Portal Proxy

Resource Description
post /fusor/api/customer_portal/login Log in to customer portal (creates session)
post /fusor/api/customer_portal/logout Log out of customer portal (deletes session)
get /fusor/api/customer_portal/is_authenticated Is there an active authenticated session to the customer portal
get /fusor/api/customer_portal/users/:login/owners (Proxied API method) Get a list of subscription owners
get /fusor/api/customer_portal/pools (Proxied API method) Get a list of subscription pools
get /fusor/api/customer_portal/owners/:id/consumers (Proxied API method) Get a list of subscription consumers
get /fusor/api/customer_portal/consumers/:id (Proxied API method) Get a subscription consumer
post /fusor/api/customer_portal/consumers (Proxied API method) Create a new subscription consumer
get /fusor/api/customer_portal/consumers/:id/entitlements (Proxied API method) Get a list of subscription entitlements
post /fusor/api/customer_portal/consumers/:id/entitlements (Proxied API method) Create new subscription entitlement
delete /fusor/api/customer_portal/consumers/:id/entitlements (Proxied API method) Delete all entitlements for a subscription consumer
