Customer Portal Proxy

Capsule to Red Hat customer portal for some API methods. Refer to customer portal documentation for parameters available to each method.

Customer Portal Proxy / login

post /fusor/api/customer_portal/login
Log in to customer portal (creates session)

Params

  • username : optional [

    String

    ]

    Customer portal account username to log in

  • password : optional [

    String

    ]

    Customer portal account password to log in

Customer Portal Proxy / logout

post /fusor/api/customer_portal/logout
Log out of customer portal (deletes session)

Customer Portal Proxy / is_authenticated

get /fusor/api/customer_portal/is_authenticated
Is there an active authenticated session to the customer portal
